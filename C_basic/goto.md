## goto문
  * 무조건 분기문
  * goto문을 만나면 바로 넘어감
    - 필요한 부분에서만 사용되고 기본적으로는 권장되지 않음
    - 변수가 많기 때문
    - 복잡한 코드를 벗어나는데는 좋지만, 자주 사용할 경우 프로그램이 꼬여버림 -> 스파게티
  * 예시
    - 코드가 복잡하게 여러 블록으로 이루어질 경우
    - 블록을 빠져나오려면 순차적으로 빠져나와야한다
    - 하지만 goto를 이용할 경우 빠져나오는 과정 없이 바로 원하는 목적지로 이동할 수 있다.
