  * 단위 표현
    - 1 byte = 8비트 = **256 경우의 수
    - 2 byte = 16비트 = **65536
    - 4 byte = 32비트 = 42.9억
    - 8 byte = 64비트 = 1844경

   * byte: 메모리, 저장창치 용랑 단위
     - 2<sup>10</sup> = 1K
     - 2<sup>20</sup> = 1M
     - 2<sup>30</sup> = 1G
     - 2<sup>40</sup> = 1T
   
   * 한글 한 문자는 2바이트 필요
