## 다른 종료 메서드
  * exit(0);
    - 프로그램 종료
    - 소속되어 있는 블럭을 빠져나가는 break와 달리 프로그램 자체를 종료함.
  * return 0;
    - 나를 호출했던 곳으로 되돌아가라
    - main 함수일 경우 종료를 의미함
