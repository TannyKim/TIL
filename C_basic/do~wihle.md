**do~while
  *![image](https://user-images.githubusercontent.com/98008421/167149043-ad6fbb3c-f3f6-49a5-9709-f716e9746031.png)
  * 조건이 뒤에있다.
    - 조건이 거짓이어도 반드시 한번은 수행한다.
    - while과 for문은 조건 뒤에 세미콜론이 없지만, do~while문은 세미 콜론이 있다.
  * while / for 과의 비교
    - 조건이 처음부터 거짓일 때,
    - while -> 한번도 수행 안함
    - for -> 한번도 수행 안함
    - do ~ while -> 일단 한번은 수행
