* &: 주소연산자
  ```c
  int a = 20;
  ```
  - &a: a의 시작 주소, a의 시작 번지

* **c 언어에서는 메모리 번지도 Data이다
 - 메모리 번지수의 범위?
   + int일 경우 -> 4byte -> 범위: 0 ~ 42.9억
   + 번지 수에 '-(마이너스)' 는 없음.

* 주소는 내부적으로(실제로는) 16진수로 저장되고 표기된다.
  - 16진수 표기 예시: 0X12F4 
  - 16진수란? 0 ~ 9, A ~ F 를 사용하여 표시
  -  printf("&a=%p", &a);

* 이해를 돕기위해 10진수로 주소를 표기한다
  - printf("&a=%u", &a);
  - u 는 unsigned -> 번지수는 음수가 안되기 때문.
